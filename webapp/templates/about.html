About

text summarization and chatbot application using nltk and spacy for data processing, and tensorflow torch and hugging face for modiling Our Search ,and flask and sqlalchemy for back end and HTML CSS JS and bootstrap for UI.

the data which used in the chatbot is a JASON file contains patterns talk about AI applications and computer science. for text summarization the data is buit in t5-large transformer model

in chatbot we used sequential model and built RNN as similarly a multi classification problem, in text summarization we used t5-transformer model as text to text problem
